## Atlas Next JS
A mock website for Atlas School students to manage their courses. It feature NextJS, React, and Node Js for a responsive website. 

# Learning Objectives
- Use NextJs routing with dynamic params to update content dependant components and perform content dependant state management. 
- Further understanding on "Use Server" directive for database fetching/posting and static generation and "Use Client" for dynamic content.
- Create OAuth authorization for security and user specfic content.
